Über Meteoino

Meteoino ist ein Schulprojekt im Rahmen des Moduls XXX, in welchem wir Mithilfe eines Mikroprozessors ein eigens Projekt verwirklichen sollen.

Bei Meteoino handelt es sich um eine Applikation, bei welcher Wetterdaten in Echtzeit auf der Website angezeigt werden. Diese Wetterdaten werden dabei von zwei einzelnen Arduinos, ausgestattet mit einem Wettersensor, Ortsunabhängig auf eine API, welche im Backend läuft, gesendet. Von dieser API aus, wird vom Frontend, sprich dieser Website, API requests an das Backend gemacht, wobei dann die letzten Wetterdaten angezeigt werden.

Durch diese drei Bestandteile sind alle Module, unabhängig von den anderen, beliebig erweiterbar. So kann zum Beispiel ein weiteres Arduino an einem anderen Standort platziert werden.
